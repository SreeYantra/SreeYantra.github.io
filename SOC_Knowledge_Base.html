OSI Layers

The OSI (Open Systems Interconnection) model is a conceptual framework that helps understand how different layers in a network communicate. It consists of seven layers: Physical, Data Link, Network, Transport, Session, Presentation, and Application. Each layer has its own function, and together, they allow data to be transferred across networks.

Networking

Networking involves connecting computers and devices so they can share information. It uses protocols like TCP/IP to ensure data is sent correctly over the internet. Networking includes understanding how devices communicate with each other, how IP addresses are assigned, and how data travels through cables and wireless networks.

Servers & Endpoints

Servers are powerful computers that store data and make it available to other devices (endpoints) like laptops or phones. Securing servers and endpoints is vital to prevent hackers from accessing sensitive information.

DNS & URLs

DNS (Domain Name System) is like the internet's phonebook. It translates human-readable website addresses (URLs) like 'example.com' into machine-readable IP addresses that computers use to locate the site on the internet.

DHCP & IP Addresses

DHCP (Dynamic Host Configuration Protocol) assigns unique IP addresses to devices on a network. An IP address is like an identification number for a device, allowing it to communicate over the internet.

Cybersecurity Fundamentals

Cybersecurity fundamentals include the CIA Triad (Confidentiality, Integrity, Availability), which ensures that information is kept safe, accurate, and accessible. OWASP Top 10 lists the most critical web application security risks, like SQL injection and cross-site scripting.

Incident Response

Incident Response is the process of identifying, managing, and mitigating cybersecurity incidents like data breaches or system hacks. A good incident response plan helps reduce the impact of attacks and restores operations quickly.

Common Cyberattacks

Cyberattacks come in many forms: DDoS (Distributed Denial of Service) attacks overwhelm websites, phishing attempts trick people into sharing personal information, and malware is harmful software that infects systems.

Email Security Investigation

Email security involves checking for signs of phishing or malware in emails. Protecting email accounts includes using strong passwords, encryption, and recognizing suspicious messages.

Network Traffic Investigation

Network traffic investigation helps identify unusual or malicious activities on a network, such as unauthorized data transfers or attacks. Monitoring tools like firewalls and Intrusion Detection Systems (IDS) are used to detect such threats.

Malware Investigation

Malware investigation involves analyzing harmful software to understand how it infects systems. Once detected, security teams remove malware and prevent future infections using antivirus programs and other tools.

Data Loss Prevention (DLP)

DLP refers to techniques used to prevent sensitive data from being accidentally or intentionally leaked out of an organization. It includes monitoring data transfers and blocking unauthorized access to critical information.
